# STM32 Ackermann Car

STM32-based 4-wheel Ackermann car with UART control using HC05 bluetooth module and obstacle detection.

## Hardware
- STM32F103C8T6
- DC motors
- IR sensor (HW201)
- HC05

## Features
- UART command control
- Garbage Data Safety Protocol
- Differential steering logic
